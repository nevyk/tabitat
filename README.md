# Tabitat

A habitat for your tabs.

Tab icons created by strip - [Flaticon](https://www.flaticon.com/free-icons/tab)

## Development

#### Requirements

| Dependency | Version |
| ---------- | ------- |
| `NodeJS`   | 16+     |
| `pnpm`     | 7+      |

## Build Instructions

The output zip file will be in the `dist` folder.

```
pnpm install
pnpm run pack
```
