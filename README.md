# Tabitat

A habitat for your tabs.

Tab icons created by strip - [Flaticon](https://www.flaticon.com/free-icons/tab)

## Development

#### Requirements

| Dependency | Version |
| ---------- | ------- |
| `NodeJS`   | 16.17.0 |
| `pnpm`     | 7.17.1  |

## Build Instructions

Build output goes to dist folder.

```
pnpm install
pnpm run build
```
